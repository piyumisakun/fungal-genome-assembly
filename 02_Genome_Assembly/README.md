# Genome Assembly

## Overview

This section describes the genome assembly workflow used to reconstruct the genome of *Perenniporia cf. tephropora* DD18 from Illumina paired-end sequencing data. The workflow consisted of de novo genome assembly using SPAdes, followed by scaffolding with SSPACE to improve assembly continuity. Additional filtering and polishing steps were performed to generate a high-quality draft genome assembly suitable for downstream analyses.

---

## Objectives

- Assemble high-quality Illumina paired-end sequencing reads into genome contigs.
- Improve assembly continuity through scaffolding.
- Generate a reliable draft genome assembly for downstream genome annotation and comparative genomics.

---

## Input Data

- Forward paired-end reads (`DD18_trim_1.fastq.gz`)
- Reverse paired-end reads (`DD18_trim_2.fastq.gz`)

# SPAdes

## Purpose

SPAdes was used to assemble quality-assessed Illumina paired-end sequencing reads into contiguous genome sequences (contigs).

### Software

- SPAdes v3.15.5

### Why was SPAdes used?

SPAdes is a de Bruijn graph-based genome assembler designed for assembling short-read sequencing data. It generates high-quality contigs that serve as the foundation for downstream scaffolding.

### Representative Command

```bash
spades.py \
-1 DD18_trim_1.fastq.gz \
-2 DD18_trim_2.gz \
-t 4 \
-m 16 \
-o spades_output
```

# SSPACE

## Purpose

SSPACE Basic v2.1 was used to improve genome assembly continuity by linking contigs into scaffolds using paired-end sequencing information.

### Software

- SSPACE Basic v2.1
- Bowtie v1.3.1
- Bash
- SeqKit

### Why was SSPACE used?

Genome assemblies generated by SPAdes often consist of numerous contigs. SSPACE uses paired-end read information to identify relationships between contigs and joins them into longer scaffold sequences, thereby improving assembly continuity.

### Workflow

1. Validate paired-end FASTQ files.
2. Split large FASTQ files into smaller paired chunks.
3. Automatically generate `library.txt`.
4. Map paired-end reads to contigs using Bowtie.
5. Scaffold contigs using SSPACE.
6. Remove scaffolds shorter than 1000 bp using SeqKit.
7. Polish the filtered assembly using Pilon.

### Representative Command

```bash
perl SSPACE_Basic.pl \
-l library.txt \
-s contigs.fasta \
-k 5 \
-a 0.7 \
-T 12 \
-b sspace_output
```

### Output

- Scaffolded genome assembly
- Filtered scaffold assembly (>1000 bp)
- Polished genome assembly
